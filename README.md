# Walk On Water

A Slime Rancher 2 mod that allows Beatrix to walk on water.

## Installation

1. Install [Melon Loader](https://melonwiki.xyz/#/?id=automated-installation) for Slime Rancher 2
1. Download the [latest release](https://github.com/markekraus/SR2WalkOnWater/releases)
1. Copy `WalkOnWater.dll` to the `Mods` folder in your Slime Rancher 2 install folder (i.e. `C:\Program Files (x86)\Steam\steamapps\common\Slime Rancher 2\Mods`)
